# Оптимизация сети пунктов выдачи заказов

## Описание
Проект по оптимизации размещения пунктов выдачи заказов (ПВЗ) в крупном городе.
Целью является сократить издержки на доставку, улучшить покрытие сервисом и оптимизировать зоны обслуживания.

## Структура проекта
- `config.json` — входные данные (районы, заказы, существующие ПВЗ, параметры)
- `main.py` — основная программа
- `utils.py` — вспомогательные функции (расчёты расстояний, интерполяция и т.д.)
- `README.md` — этот файл

## Входные данные (config.json)
- **districts** — массив многоугольников районов города
- **historical_orders** — историческая база заказов с координатами и весами
- **existing_pickup_points** — существующие ПВЗ с пропускной способностью
- **task_parameters** — параметры задачи (количество новых ПВЗ, радиус доставки и т.д.)

## Выходные данные
Программа производит:
1. **demand_density_map** — карта плотности спроса (тепловая карта)
2. **new_delivery_points** — координаты оптимальных мест для новых ПВЗ
3. **delivery_zones** — зоны обслуживания каждого ПВЗ
4. **metrics** — метрики эффективности (среднее расстояние, покрытие, баланс нагрузки)

## Запуск
python main.py

## Использованные алгоритмы
- **K-means** — для поиска оптимальных мест размещения новых ПВЗ
- **Voronoi diagrams** — для определения зон обслуживания
- **Kernel Density Estimation** — для интерполяции плотности спроса
